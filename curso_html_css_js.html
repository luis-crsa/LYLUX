Curso de HTML, CSS e JavaScript

Ministrado pelo Professor Casseb. Aprenda a construir websites modernos e interativos com HTML, CSS e JavaScript.

Sobre o Professor Casseb

O Professor Casseb é um especialista em desenvolvimento web com mais de 10 anos de experiência. Ele é apaixonado por ensinar e já ajudou centenas de alunos a se tornarem desenvolvedores web de sucesso.

Módulo 1: Introdução ao HTML

Entenda os fundamentos do HTML e como criar a estrutura básica de uma página web.

Módulo 2: Estilização com CSS

Aprenda a estilizar suas páginas e torná-las responsivas para diferentes tamanhos de tela.

Módulo 3: Fundamentos de JavaScript

Descubra como adicionar interatividade e lógica aos seus sites com JavaScript.

Módulo 4: Layouts Responsivos com Flexbox e Grid

Explore técnicas para criar layouts flexíveis e responsivos usando Flexbox e Grid.

Módulo 5: Interatividade com JavaScript

Aprofunde-se na criação de interações dinâmicas e manipulação de eventos com JavaScript.

Módulo 6: Projeto Final

Coloque em prática tudo o que aprendeu desenvolvendo um projeto final completo.
